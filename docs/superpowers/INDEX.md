**brainstorm ‚Üí design spec (`specs/`) ‚Üí implementation plan(s) (`plans/`) ‚Üí execute & merge**.
Plans use `- [ ]` checkboxes for task tracking ‚Äî boxes are intentionally left unchecked
in-file. Every cycle runs a `feature-dev:code-reviewer` pass (often a review LOOP until no
Critical/Important remain) before a `--no-ff` merge to `main`.

Files are dated `YYYY-MM-DD` by authoring date. **All work below is merged to `main`.**
This index is a navigation aid ‚Äî open the individual file for full detail; don't
consolidate these into one document (it would destroy the dated provenance). Run
`ls specs/` / `ls plans/` for exact filenames; paths below are abbreviated with `‚Ä¶`.

| **47c ‚Äî Nested symbols: INDEPENDENT PER-INSTANCE TIMELINES** (each instance runs its own internal clock ‚Üí two instances of one symbol show DIFFERENT frames at the same wall-clock moment, the real Flash-MovieClip payoff). PIVOTAL: the engine seam was ALREADY there ‚Äî `flattenInstances` threads a `localTime` down its recursion and samples every leaf at it (in 47a‚Äìedit it was just the global time); 47c replaces the ONE identity line with a per-instance REMAP. New optional `SceneObject.symbolTime?: { startOffset; loop; speed }` ‚Äî **ABSENT = identity remap**, the parity-safety mechanism (existing projects + the parity test are byte-unchanged; a fresh instance stays in lockstep until the user opts in). `remapLocalTime(parentTime,timing,symbolDuration)` = shift by startOffset, scale by speed, hold-first-frame before start (`t<=0‚Üí0`), then LOOP (`t%dur`, t>0 guaranteed so no negative-mod) or ONE-SHOT (`min(t,dur)`); `symbolDuration<=0‚Üí0` (no NaN).
